Layered Geometry and Parallax
Rendering fur or other complex bumpy surfaces is challenging. As the material involves depth, meaning it can't be approximated by a flat surface. But it is possible. For example Red Dead Redemption 2 does an exceptional job rendering fur, and while playing it I got inspired. So I went down a rabbit hole trying to implement my own fur-like shading.
Shell Shading
One way to render depth, is to literally stack multiple surfaces. This technique is called shell shading (not to be mixed up with cell shading), and it has been used for decades. So I decided to use Blenders geometry nodes to put together a basic implementation.
Shell shading works quite well for rendering fur-like surfaces. I even went on to program a shader in Unity which creates this same effect more efficiently.

But it does have drawbacks, such as it is considerably more GPU intensive then a flat surface. It's a illusion also fails when viewed at harsh angles. So I did some more research to find other ways to render a fur-like material...
Parallax Mapping
Parallax mapping creates depth by shifting where the textures are sampled. It uses the incoming camera ray to calculate the parallax causes by the viewing angle.

This means instead of rendering a material across multiple triangles to create depth, it renders one triangle and calculates how depth would effect its appearance.


Version 1
